# Basic ETL Pipeline (Python + SQL)

This project demonstrates a simple ETL (Extract, Transform, Load) pipeline.

## 📌 Steps Performed:
- Extracted data from a CSV file
- Transformed data by cleaning missing values using Pandas
- Loaded cleaned data into a SQLite database

## 🛠 Tech Stack:
- Python
- Pandas
- SQLite
- Git & GitHub

## 🎯 Purpose:
To understand basic data engineering concepts and data processing workflow.

## 🚀 Output:
- Cleaned dataset
- Stored data in database for querying
